This extension reads a CSV file, matches each student ID on the page, fills the score field, and updates the selected reason code for all students.

Quick steps:

  1. Open the marks page in Chrome.
  2. Select your CSV file.
  3. Choose the ID column and score column.
  4. Choose one reason code.
  5. Click Fill Scores.

Example CSV:

id,score
202303387,92.14
202303552,85.71

Tips:

  • The first row should contain column names.
  • If your Excel sheet has other columns, just choose the correct ID and score columns after upload.
  • The preview shows the first 10 matched rows from the file.
  • If a student ID is missing on the page, that row will be counted as not found.
